# Balance
Units in this mod are balanced in a popular way so that there's always counter to one or another unit, like rock paper scissors
![Mod Balance](https://github.com/Falkowitz/Medieval-Mindustry/assets/119055662/33a2cb1a-0b1d-4c1d-90fc-549c96d9428a)

This applies to the main units tree with the general idea that shields counter ranged, swords counter shields and ranged counter swords
